# C# Tutorial Project

This is a C# tutorial project for you to learn C# in a comprehensive way.

## Overview

Welcome to my project! This README file provides you the comprehensive details about the project, its purpose, functionality, setup instructions, and usage guidelines for learning C# from the beginning to advance.

My project is designed for people who want to learn C# with or without any experiences. It incorporates C# version 9.0.311, Visual Studio Code, and Git.

## Prerequisites

Before you begin, ensure you have met the following requirements:

1. C# installer installed on your system whichever version you can use. In this project, the contributor uses version 9.0.311. [Click here to download](https://dotnet.microsoft.com/en-us/download/dotnet/9.0).
2. Visual Studio Code for streamlining code editor with support for development operations like debugging, task running, and version control. [Click here to download](https://code.visualstudio.com/download).
3. Git for managing source code and repository. [Click here to download](https://git-scm.com/downloads).

## Installation

Follow these steps to set up the project on your local machine:

Open your local directory (e.g., \Documents) and clone the repository:

```bash
git clone <repository_url>
```

Navigate to the project directory:

```bash
cd <project_directory>
```

Compile the source code:

```bash
csc <file_name>.cs
```

Run the application:

```bash
./<file_name>
```
